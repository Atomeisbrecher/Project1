Interactor = Use Case, т.е. имплементация конкретного случая использования какого-то элемента приложения.

Для этого нам нужны логика работы базы данных, логика работы с моделью данных, и логика работы с каким либо ещё функционалом (интерфейсы).

В интерфейсы передаются dto - дата классы, описывающие что передаётся в параметры интерфейса.

from typing import Protocol - элемент в питоне, созданный для описания интерфейсов
